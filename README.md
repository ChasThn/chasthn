# L'Odyssée d'un Bit

## Présentation

Ce dépôt présente un ensemble de programmes que j'ai écrits sur différents thèmes dans
la perspective d'incrémenter mes compétences et mes connaissances sur les ordinateurs
et les sujets abordés. Il sera mis à jour à chaque étape d'avancement d'un projet.

## Thèmes

Les thèmes traités sont les suivants :

- L'atelier
- Les jeux
- Les simulations
- La collecte d'informations
- L'intelligence artificielle
- Les grands nombres
